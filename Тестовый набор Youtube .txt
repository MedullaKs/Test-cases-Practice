Тест кейс №1
Переход во вкладку Shorts с главной страницы сайта


Шаги
	Ожидаемый результат

Тест кейс №2
Возможность поделиться ссылкой на видео через “Вконтакте” на стене профиля


Шаги
	Ожидаемый результат
	1.В левом верхнем углу страницы кликнуть на иконку боковой панели сайта (три полоски)
	1.Разворачивается боковая панель сайта с вкладками, подписками и плейлистами
	2.Кликнуть по вкладке Shorts
	2.Происходит переход на страницу вкладки Shorts
	3.В правой нижней части первого видео нажать на иконку “Поделиться”
	3.Возникает всплывающее окно с текстом ссылки и иконки различных мессенджеров, где можно поделиться ссылкой 
	4.Кликнуть по иконке “Вконтакте”
	4.Происходит перенаправление на сайт “Вконтакте”, по умолчанию выбрана аудитория “Друзья и подписчики”, по умолчанию выбрана опция “Добавить изображение”
	5.В правом нижнем углу экрана нажать мышкой по кнопке “Отправить”
	5.Появляется сообщение “Поздравляем! Ссылка появится на вашей странице.Это окно закроется через несколько секунд”
	6.В профиле “Вконтакте” перейти на режим просмотра стены (опция “Моя страница” в боковой панели)
	6.Последняя опубликованная запись на стене профиля - ссылка на видео в Youtube
	7.Кликнуть по ссылке Youtube, опубликованной на стене профиля
	7.Видео воспроизводится на стене профиля 
	8.Кликнуть по иконке Youtube внутри воспроизводящегося видео “Смотреть это видео на youtube.com”
	8.Ссылка корректно открывается в новой вкладке в браузере по умолчанию, видео начинает воспроизводиться
